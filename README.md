# Sports Equipment Rental Management System

## Опис проекту
Система управління прокатом спортивного спорядження, розроблена під час технологічної практики для фітнес-центру "СпортЛайф". Додаток дозволяє автоматизувати облік інвентарю, клієнтів та операцій оренди, забезпечуючи зручний інтерфейс для адміністраторів.

## Основні функції
- **Управління спорядженням**: додавання, редагування, видалення, пошук і фільтрація.
- **Клієнтська база**: реєстрація клієнтів, перегляд історії оренд.
- **Оренда**: оформлення прокату, автоматичний розрахунок вартості, контроль термінів повернення.
- **Повернення**: автоматичне оновлення статусу та доступності спорядження.
- **Звіти**: відображення активних прокатів, індикація прострочень, статистика доступності.

## Встановлення (через інсталятор)
1. **Завантажте інсталятор** з релізної сторінки або архіву проекту.
2. **Запустіть інсталятор** (`SportsRental_Setup.exe`).
3. Дотримуйтесь інструкцій майстра встановлення:
   - Виберіть папку для встановлення.
   - Дозвольте встановлення необхідних компонентів (.NET Framework, SQL Server Express).
4. Після встановлення ярлик програми з'явиться на робочому столі та в меню "Пуск".
5. Запустіть **SportsRental** — система автоматично налаштує базу даних при першому запуску.

> **Примітка**: Якщо SQL Server Express вже встановлений, інсталятор використає існуючий екземпляр або створить новий з іменем `SQLEXPRESS02`.

## Інструкція з використання
### Додавання спорядження
1. Перейти на вкладку **"Спорядження"**.
2. Заповнити поля: назва, категорія, рік виготовлення, бренд, кількість.
3. Натиснути **"Додати"**.

### Оренда спорядження
1. Вибрати спорядження зі списку.
2. Натиснути **"Орендувати"**.
3. У вікні обрати клієнта, вказати кількість днів.
4. Підтвердити оренду.

### Реєстрація клієнта
1. Перейти на вкладку **"Клієнти"**.
2. Натиснути **"Додати клієнта"**.
3. Заповнити ПІБ, телефон та email.
4. Підтвердити додавання.

### Повернення спорядження
1. Перейти на вкладку **"Прокат"**.
2. Вибрати активний прокат.
3. Натиснути **"Повернути"**.
4. Підтвердити повернення.
